📘 Trabajo Práctico N°1 – JavaScript

Taller de Programación III   
Departamento de Informática – EEST N°1 "Raúl Scalabrini Ortiz"

🧩 Descripción

Este trabajo práctico introduce los conceptos fundamentales de JavaScript, a través de ejercicios progresivos que combinan:

Declaración de variables
Operaciones matemáticas
Uso de funciones
Lógica condicional
Manipulación de strings

El objetivo es construir una base sólida en programación, desarrollando habilidades esenciales para resolver problemas computacionales.

🎯 Objetivos de Aprendizaje

Al completar este práctico, se adquieren las siguientes competencias:

🔹 1. Manejo de variables y tipos de datos
Declaración de variables con const
Uso de tipos primitivos:
string
number
boolean
Comprensión del tipado dinámico en JavaScript
🔹 2. Operadores y expresiones
Operadores aritméticos:
Suma (+)
Resta (-)
Multiplicación (*)
División (/)
Módulo (%)
Evaluación de expresiones lógicas (===)
Resolución de problemas matemáticos simples
🔹 3. Funciones
Definición de funciones con parámetros
Uso de return para devolver resultados
Diferencia entre mostrar (console.log) y retornar valores
Reutilización de código
🔹 4. Lógica y comparaciones
Uso de operadores de comparación:
Igualdad estricta (===)
Mayor que (>)
Menor que (<)
Construcción de funciones booleanas
Evaluación de condiciones
🔹 5. Manipulación de strings
Concatenación de textos
Uso de variables dentro de strings
Creación de mensajes dinámicos
🔹 6. Matemática aplicada en programación
Potencias (cuadrado, cubo, exponentes)
Redondeo de números:
Math.round()
Math.ceil()
Cálculo de resto (módulo)
Cálculo de áreas geométricas simples
🛠️ Habilidades Desarrolladas

Durante la resolución de los ejercicios, se fortalecen habilidades clave:

✔ Pensamiento lógico
✔ Resolución de problemas paso a paso
✔ Lectura e interpretación de consignas técnicas
✔ Escritura de código claro y estructurado
✔ Identificación de errores comunes
✔ Abstracción mediante funciones
📌 Estructura del Trabajo

El práctico se divide en dos grandes bloques:

🧪 Parte 1: Variables y expresiones
Creación de variables
Resolución de operaciones matemáticas simples
⚙️ Parte 2: Funciones
Implementación de funciones para:
Operaciones matemáticas
Comparaciones
Manipulación de strings
Cálculos geométricos
🚀 Requisitos
Editor de código (VS Code recomendado)
Navegador web o entorno Node.js
Conocimientos básicos de programación (introductorios)
📚 Conclusión

Este trabajo práctico permite dar los primeros pasos en JavaScript, consolidando los fundamentos necesarios para avanzar hacia temas más complejos como:

Estructuras de control (if, for, while)
Arrays y objetos
Programación orientada a objetos
Desarrollo web
